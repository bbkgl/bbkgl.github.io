# 斐波那契数列

---

C++。这题真的没什么好说的了。。。

公式：`f(n) = f(n-1) + f(n-2)`，边界：`f(0) = 0, f(1) = 1`

代码如下：

```cpp
class Solution {
public:
    int Fibonacci(int n) {
        int dp[40];
        dp[0] = 0;
        dp[1] = 1;
        for (int i = 2; i <= n; i++) {
            dp[i] = dp[i-1] + dp[i-2];
        }
        return dp[n];
    }
};
```
